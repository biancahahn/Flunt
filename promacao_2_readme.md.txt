O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, 
o famoso criador do kernel do sistema operacional Linux. Um número impressionante de projetos de software 
depende do Git para controle de versão, incluindo projetos comerciais e de código-fonte aberto.

Qual a função do Git?
Git é um sistema de controle de versão de arquivos. Através deles podemos desenvolver projetos na qual 
diversas pessoas podem contribuir simultaneamente no mesmo, editando e criando novos arquivos e 
permitindo que os mesmos possam existir sem o risco de suas alterações serem sobrescritas..
